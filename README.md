# Implementasi-Interpolasi-Lagrange-Orde-3-untuk-Aproksimasi-Fungsi-
Mini project metode numerik menggunakan Interpolasi Lagrange orde-3 untuk melakukan pendekatan nilai fungsi dari data diskrit.  Dalam project ini dilakukan proses pembentukan polinom Lagrange dari beberapa titik data untuk mengestimasi nilai fungsi pada titik tertentu menggunakan Microsoft Excel.
